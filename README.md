# psi-binning

Песочница для **бининга фичей** и расчёта **PSI** (Population Stability Index) — мониторинга
стабильности распределений по периодам. Биннинг отделён от PSI и переиспользуем (напр. для WoE).

## Структура

| Файл | Назначение |
|------|------------|
| `binning/` | `NumBinner` (числовой) и `CatBinner` (категориальный) — fit/transform в стиле sklearn. Идентичность бина pandas-нативная: интервал `pd.Interval`, точка `float`, служебные `other`/`missing`. |
| `psi.py` | `define_base_data` (выбор базы) → `calc_psi_by_features` / `calc_psi_by_period`; PSI из counts со сглаживанием Лапласа. |
| `psi_plots.py` | Графики plotly: доли бинов по периодам + PSI с порогами 0.10 / 0.25. |
| `synthetic_data.py` | Генерация выборки (12k строк × 12 месяцев, 7 фичей под краевые случаи). |
| `build_notebook.py` | Сборка демо-ноутбука `psi_playground.ipynb`. |

## Запуск

Проект на [uv](https://docs.astral.sh/uv/) (Python 3.14):

```bash
uv sync
uv run python build_notebook.py   # пересобрать ноутбук
```

Открой `psi_playground.ipynb` (kernel «Python 3.14 (.venv)») и выполни ячейки сверху вниз.

## Подход

- **Точки + интервалы:** супервстречаемые значения (доля ≥ `point_share`) получают свой точечный
  бин, и интервал их не накрывает; мало уникальных → дискретный режим; редкие интервалы сливаются.
- **PSI** = Σ (aᵢ − eᵢ)·ln(aᵢ / eᵢ); пороги: < 0.10 стабильно, 0.10–0.25 сдвиг, > 0.25 сильный.
  Сглаживание Лапласа убирает деление на ноль на пустых бинах.
- **База (reference)** выбирается окном по дате (`base_size` / `shift_size`) или готовой маской.
